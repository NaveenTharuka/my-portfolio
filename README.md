# 🌐 Personal Portfolio - Frontend

A modern, responsive developer portfolio built with React to showcase my projects, technical skills, certifications, and professional journey. The application features smooth animations, a clean UI, and communicates with a FastAPI backend to manage portfolio content dynamically.

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern UI with smooth animations
- Dynamic project showcase
- Skills and technologies section
- Certificates & achievements
- Contact section
- Admin dashboard integration
- REST API integration with FastAPI backend

## 🛠️ Tech Stack

- NextJS
- Vite
- JavaScript
- Tailwind CSS
- Axios
- Next Router
- Framer Motion

## 📂 Project Structure

```
src/
├── components/
├── pages/

├── hooks/
├── assets/
└── utils/
```

## 🚀 Getting Started

### Clone the repository

```bash
https://github.com/NaveenTharuka/my-portfolio.git
```

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:3000
```
